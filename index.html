总会有些许困惑

发表于 2025-02-07 分类于 宇
本文字数： 913 阅读时长 ≈ 3 分钟

自从上次和宇见面之后似乎和她聊得更多了，破冰？大概吧。

有时发现宇对待事情也是挺认真的，下一天要去上班头晚上能把红色的甲片给取了

大概也就是从那天晚上开始，宇开始发给我她随手拍到的东西以及路边的卡通化的包子

说实话上次见面第二天宇和我主动聊天之后我总想着会不会是昙花一现，但我似乎是多想了，她还是会主动的分享之前我们聊到的事情，就比如之前谈到她的新工作

说实话，那天真的惊到我了，宇分享起来那个活力有时我都接不住，但是这样充满活力的人真的让人着迷！现在回过头来看一看似乎整个本科生涯中我遇到的人中都没人能有她这样充满活力的，也只有那时的我自己才能和她拼一拼，可惜一切都不复从前了，向往且热爱生活的我似乎倒在了那个点上。

宇有时也会分享给我她感兴趣的东西

后来她发给我看了音乐节和酒吧里的视频，我发现这个女孩子好社牛（笑！）能和不认识的人玩在一起，我以前只有在个人的圈子里才能达到她这样的状态

有时发现宇发消息的时候特别喜欢发~，像习惯性发的，总让我捉摸不透！

我发现自己似乎并不擅长聊天和接话，总感觉话题会被我一不小心聊死，每次都是宇总能及时把它拉回。有时，我会感到一种无力感，害怕自己没法把握住这份关系，害怕有一天宇不再愿意再伸出手，或者我们之间的对话会突然变得无话可说。毕竟，时间终究会磨灭一切，也许有一天，我的博客里再也没有“宇”这一分类。每次写到这里，我都会感到一丝遗憾，心里也难免有些不甘。但到那时，我或许早已释然，我会告诉自己，“这一切，都是我自找的。”也许那时，我的生活会悄然恢复平静，心中再也不会有那份悸动了。

说实话，我现在还不确定自己对她的感觉到底是喜欢她，还是喜欢她那份对生活的热爱，抑或是我陷入了某种思维定势——毕竟是相亲介绍的。虽然天使曾跟我说过，这不是让我们去相亲，而只是希望我们加个微信，相互认识一下。

我想，我对宇的感觉可能是杂糅在一起的，命运总是这么捉弄人。也许没有天使的介绍，我和宇一辈子都不会认识，就算偶然碰面，也不过是点头之交。甚至，或许她最后会成为别人的女孩。如果宇没有展现出她的热情和对生活的热爱，我或许也没有勇气去继续接触下去，毕竟在我眼中，她一直是那么的优秀。

别贪心，我们不可能什么都拥有；也别担心，我们不可能什么都失去。就算最后宇不再挽救我们，选择了更好的生活，我也会珍藏这段难忘的回忆。

更换图床

发表于 2025-02-05 分类于 生活
本文字数： 310 阅读时长 ≈ 1 分钟

今天看到几个站点照片被盗刷的帖子，一夜就被盗刷50G又或者1.7T，最后欠费一堆变成一个纯纯的大冤种，吓得我连忙看了看我的阿里云oss，之前图方便想着直接用hexo部署图片每次部署的时候都要花大量的时间还不如搞个阿里云的图床，这样不仅快而且还方便。现在直接把阿里云的oss给设置成私有。然后转过头来继续使用hexo部署。但是修改那些md文件的发现，我之前使用oss是通过PicGo上传然后用链接直接访问，那为什么我不直接丢在git上然后直接读链接呢？虽然hexo也是托管在git上，但是调用链接和每次都把所有的图片部署一遍的时间是完全不一样的，最后当然是我憨出出的又把md文件全修改一遍，弄了之后发现git链接弄出来全是裂图，然后又回到最开始的地方，继续使用hexo部署图片，好在我现在站点博文不多，图片也少，记录一下憨包的一天。

冬梅姐订婚，大年初五一家人进凤庆。

我一开始在想会不会时间太紧了，毕竟打算这初五进去、初六订婚、初七早上直接回孟定。好不容易进凤庆了想着约宇见一面，但又怕时间太紧匆匆忙忙留下的印象不太好。经过一番思想斗争还是决定发消息给宇约着她见一面！

我算是幸运的没被拒绝，现在回过头看，还得感谢宇给了这次机会

有意思的是

第一次date的地方是乐品温酒馆！

date结束，我欠宇六杯酒，她欠我一皮条

也许是我的错觉，感觉这次date之后相互了解得更多了一些，和宇的聊天变得频繁多了

之后发现宇是个感性的女孩子

也是一个热爱记录生活的人！

在我眼中，宇是一个充满活力的人，似乎从不为局促和不安所困扰。即使时间的流逝与生活的琐碎在她周围交织，她那份对自我追求的坚持和对生活的热爱依然坚定不移。她散发出的自信与优雅，如同盛开的花朵，吸引着每一只停驻其旁的彩蝶。而我，也在她的光芒下悄然驻足，心中难掩那份由衷的倾慕，仿佛她的存在是那最美的花朵，令人不自觉地被吸引，无法移开视线。

不知下次相聚又是何时，不知未来是什么样！

24年度总结

发表于 2025-01-28 分类于 年度总结
本文字数： 528 阅读时长 ≈ 2 分钟

24年对于我来说算是过去了，24年一整年算是很折磨了，特别是科研方面，但是好在我慢慢适应了下来只不过碰到事情还是会炸，就像猫应激一样。这一年等TVT的消息等得我难受，特别是有一段时间每天晚上都会惊醒来看邮箱。虽然现在已经收到消息了，TVT大修，但是返修了之后还是没消息，难受啊到现在还没有达到毕业要求，烦！

为了定选上岸，公基的题刷了2w+，最后在一个收拾好心情准备去走一走中山陵的星期天下午知道了定选修改题型增加了行测题，我不知道到时候能不能上岸了，本来就有把握上岸了，这么一搞直接没信心了，转过头来学行测发现自己不是学行测的那个料。只能笨鸟先飞慢慢搞了，一切听天由命。

阿婆今年住院了好几次，看着也是替她难受，有时候气啊，我没法做到很好的控制我的情绪。但是阿婆总是不听话啊。哎！但愿阿婆长命百岁，福寿如山！

家里来了新成员——钱宝！一只可爱的小豹猫，特别喜欢但是有点野。

最后最后，年前一点点时间，小姨妈和我妈给我介绍了个女孩子，说是相互认识认识，这个女的是小姨妈他们去东北旅游发现性格特别好的，介绍给我认识认识。除夕夜加上微信，但是感觉我好像加点没加在聊天上？又或者我没那个实力，没事这种东西只能随缘，但我总感觉我差了点什么东西，自己知道但是又不想去想。

二四年就这么过去了，但愿我二五年能够上岸定选吧，我真的太累了，我发现我已经学不动了，该休息一下了！

祝我明年万事如意心想事成！
